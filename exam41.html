태그 선택자

HTML 문서 내의 동일한 태그 모두에게 같은 스타일을 적용시키고자 할때 사용한다. 

 p {color:blue;}와 같은 형태로 사용한다.

아이디 선택자

HTML문서 내의 선택한 특정 아이디에 해당하는 요소의 스타일을 설정할때 사용한다.

 #아이디명 { color:red };

클래스 선택자

HTML문서 내에 특정 그룹을 짓는 요소들을 하나의 스타일로 묶어 선택할 때 사용한다. 

 .클래스명 { color:green };
