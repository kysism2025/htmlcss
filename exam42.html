CSS 속성 선택자

기본 선택자 뒤에[]를 사용하여 특정 속성과 해당 속성의 값을 통해 스타일을 설정할때 사용한다. 

            선택자[속성=속성값]{...}형식으로 쓰인다.
            -- 기회 별의미 --
            =  : 속성의 값이 같을 때
            ~= : 특정 단어를 속성값으로 가질때(띄워쓰기)
            ex) id~=test --> id = "test abcd qwer" ..... O
                         --> id = "test1234" .... X

            |= : 특정 단어를 속성값으로 가질때(""안에 포함되는 것 모두 확인)                         
            ^= : 속성값이 특정 단어로 시작할때
            $= : 속성값이 특정 단어로 끝날때
            *= : 속성값이 특정 단어를 포함할때

동의 선택자

동위 선택자는 동급관계에 있는 형제 요소에 스타일을 설정할때 사용한다. 

            선택자A+선택자B {...} : A 바로 뒤에 있는 B요소를 선택할 때 
            선택자A~선택자B {...} : A 뒤의 모든 요소를 선택 할 때

반응 선택자

사용자의 움직임(마우스 클릭, 마우스를 올려 놓았을 때)을 감지하여 스타일을 변경하는 선택자이다. 

            선택자:active{...} / 사용자가 마우스로 클릭했을때
            선택자:hover{...}  / 마우스 커서가 해당 요소 위에 있을 때

상태 선택자

입력 양식에 따라 변화하는 스타일을 설정할 때 사용한다. 

            input:checked{...} / 체크 상태 여부에 따른 스타일

focus(초점)가 맞춰진 상태에서 스타일 적용도 가능하다.



사용 가능 여부와 관련한 스타일 적용도 가능하다.
